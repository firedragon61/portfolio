### How to Use Recruiters for Job Search in Australia
This presentation was created with Microsoft Powerpoint. The ppt file was fed into Microsoft M365
Copilot to generate a narration script. The narration script was fed into Microsoft Clipchamp's
text-to-speech (TTS) function to generate an audio transcript. Using OBS Studio, the ppt file
presentation was captured with the audio file narrating and the final video was generated.
